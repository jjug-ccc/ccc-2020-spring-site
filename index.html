開催概要 / Description

JJUG CCCは毎年2回、春と秋に開催する日本最大のJavaコミュニティイベントです。Java関連の技術や事例に関する良質なセッションが行われ、また異なる分野で活躍するJava技術者が一堂に会する場ともなっています。ぜひご参加ください。

JJUG CCC is the largest Java community event organized by Japan Java User Group (JJUG), held twice a year in spring and autumn. It provides great sessions of Java-related technologies and practices, and brings together Java engineers from different disciplines.

JJUGとは / About JJUG

日本Javaユーザーグループ（Japan Java User Group/JJUG）は6000人以上の Javaエンジニアが参加する日本最大のJavaコミュニティです。
以下のような活動を行っています。

The Japan Java User Group (JJUG) is the largest Java community in Japan with over 9000 Java engineers.
The following activities are conducted.

ナイトセミナー / Night Seminar
毎月1回夜に開催する勉強会 / Seminar held once a month
JJUG CCC
年2回（春と秋）に開催する1日カンファレンス / Full-day conference held twice a year (Spring and Autumn)
地方講師派遣 / Instructor Dispatch
地方Javaコミュニティへの講師派遣 / Dispatch of instructors to local Java communities in Japan
国際会議参加 / Participation in international conferences
世界中のJUGとの交流 / Making a relationship with international Java Communities
